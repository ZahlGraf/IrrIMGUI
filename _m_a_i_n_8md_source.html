75  // Irrlicht OpenGL Settings

77  IrrlichtParams.DriverType = video::EDT_OPENGL;
78  IrrlichtParams.WindowSize = core::dimension2d<u32>(1024, 800);
79  IrrlichtParams.Bits = 32;

122 ### <a name="ShortUsageReferenceCustomEventReceiver"></a> Use custom Event Receiver
123 #### <a name="ShortUsageReferenceCustomEventReceiverMethod1"></a> Method 1: Inherit from standard Event Receiver
124 
125 * Inherit from the IrrIMGUI::CIMGUIEventReceiver class
126 * Overwrite method IrrIMGUI::CIMGUIEventReceiver::OnEvent to handle the event first
127 ```cpp
128 class CMyOwnEventReceiver : private IrrIMGUI::CIMGUIEventReceiver
129 {
130  bool OnEvent(const irr::SEvent & rEvent)
131  {
132  // do your own stuff with rEvent...
133 
134  // when rEvent has not been consumed by your game logic pass it to the CIMGUIEventReceiver class
135  return CIMGUIEventReceiver::OnEvent(rEvent);
136  }
137 };
138 ```
139 
140 * Don't forget to call IrrIMGUI::CIMGUIEventReceiver::OnEvent when you want to pass this Event to the GUI
141 * Give a pointer of this custom Event Receiver while creating the GUI handle object and the Irrlicht Device
142 
143 #### <a name="ShortUsageReferenceCustomEventReceiverMethod2"></a> Method 2: An Event Receiver from the scratch
144 
145 * Create an Event Receiver class that inherits from `irr::IEventReceiver` and IrrIMGUI::CIMGUIEventStorage
146 * Overwrite the `irr::IEventReceiver::OnEvent` method to handle the event
147 * If you want to pass any input event information to the GUI, just write to the corresponding variables of IrrIMGUI::CIMGUIEventStorage
148 * Give a pointer of this custom Event Receiver while creating the GUI handle object and the Irrlicht Device
